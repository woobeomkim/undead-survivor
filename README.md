# Undead Survivor

> 뱀서류(Vampire Survivors) 스타일 게임을 참고한  
> Unity 기반 클론 코딩 학습 프로젝트입니다.

---

##  프로젝트 개요

- **장르**: 2D 액션 / 생존
- **참고 게임**: Vampire Survivors
- **개발 목적**:  
  다수의 적 처리와 자동 전투 흐름을 분석하고  
  Unity에서 구현하며 게임 루프를 이해하기 위한 프로젝트입니다.
- **개발 형태**: 개인 학습 프로젝트
- **엔진 / 언어**: Unity / C#

---

## 학습 목표

- 대량의 적 처리 흐름 이해
- 플레이어 중심 자동 전투 구조 분석
- 게임 루프 단순화 및 반복 구조 이해

---

## 구현 내용

- 플레이어 이동 및 자동 공격
- 다수의 적 스폰 및 추적
- 체력 관리 및 게임 오버 처리

---

## 배운 점

- 단순한 조작 구조에서도  
  게임 재미를 만드는 루프 설계의 중요성
